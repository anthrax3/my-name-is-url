my-name-is-url

About

In a nutshell, my-name-is-url is an intelligent parser that searches a string of text for urls.

The url spec is so vague that many strings could be a url so matching the spec directly results in a lot of false positives.

The regular expression used in my-name-is-url tries to match patterns likely to represent a url in a sentence rather than matching the actual url spec.

Bugs

If you spot anything odd it would be awesome if you could submit an issue on Github with your unique url and a description of what's wrong. Thanks :)
